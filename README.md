# Exercício Python - Variáveis

Este é um exercício simples em Python para treinar o uso de:

- variáveis
- entrada de dados com `input()`
- conversão de tipos (`int` e `float`)
- saída formatada com `f-strings`

## Projeto

O programa cria uma ficha de jogador com as informações digitadas pelo usuário.

## Conceitos Praticados neste Projeto

Strings;
Números inteiros;
Números decimais;
Operações com variáveis

Exibição de dados formatados

## Como executar

1. Instale o Python
2. Baixe este projeto
3. Execute no terminal:

```bash
python ficha_jogador.py
